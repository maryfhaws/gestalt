The videogame Tetris created by Alexey Pajitnov uses color similarity to distinguish the Tetriminos, or the shapes each composed of four squares that are placed strategically during gameplay. Each Tetrimino has its own iconic color, which helps players quickly recognize its shape and determine the optimal placement.

Color similarity is used in Visual Studio Code and other coding tools to help users scan their code. For instance, all HTML tags have the same color, which is different from the color for CSS styles, which is different from the color of a JS function. The principle of similarity assists coders with identifying certain elements within their code.
